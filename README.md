# Guess the Number Game

A simple command-line game where the player tries to guess a randomly generated number between 1 and 100.

## How to Play

1. The program will randomly select a number between 1 and 100;
2. You must guess the number by typing your guess and pressing Enter;
3. After each guess, you will receive a hint:
   - `Too low` – your guess is lower than the secret number;
   - `Too high` – your guess is higher than the secret number;
4. Keep guessing until you find the correct number;
5. You can exit the game at any time by typing `exit`.

## Winning the Game

When you guess correctly, the program will display a congratulatory message showing the secret number and the number of attempts you made.

## Requirements

- Python 3.6 or higher (no external libraries are required);
- Works on Linux, macOS, Windows.

## How to Run

1. Save the code in a file, for example `guess_the_number.py`;
2. Open the terminal in the same directory;
3. Run the script: `python guess_the_number.py` or `python3 guess_the_number.py`;
4. Enjoy! And try to guess the number in as few attempts as possible.
